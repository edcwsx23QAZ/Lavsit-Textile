# Документация по поставщикам

Эта папка содержит документацию по каждому поставщику, организованную по следующей структуре:

```
suppliers/
├── {supplier-name}/
│   ├── stock-parser.md      # Правила парсинга тканей (наличие, метраж)
│   ├── price-parser.md      # Правила парсинга прайса (цены, категории)
│   ├── exclusions.md        # Исключения из парсинга
│   └── data-structure.md    # Структура данных и сохраненные файлы
```
